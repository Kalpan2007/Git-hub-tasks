#### **Task 12: Stashing Changes**
1. Save uncommitted changes:  
   ```bash
   git stash
   ```
2. Apply stashed changes:  
   ```bash
   git stash apply
   ```
3. Drop the stash:  
   ```bash
   git stash drop
   ```

 # **Task 6: Branch Management**
1. Create a branch and switch to it:  
   ```bash
   git checkout -b feature-branch
   ```
2. List branches:  
   ```bash
   git branch
   ```
3. Rename a branch:  
   ```bash
   git branch -m feature-branch feature-enhanced
   ```

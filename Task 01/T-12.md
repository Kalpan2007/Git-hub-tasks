# **Task 12: Cherry-Picking Commits**
1. Create a new branch:  
   ```bash
   git checkout -b cherry-pick-example
   ```
2. Cherry-pick a specific commit from another branch:  
   ```bash
   git cherry-pick <commit-hash>
   ```

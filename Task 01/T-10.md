# **Task 10: Using Git Stash**
1. Make changes to a file but donâ€™t commit:  
   ```bash
   echo "Temporary work" >> temp.md
   ```
2. Stash the changes:  
   ```bash
   git stash
   ```
3. View stashed changes:  
   ```bash
   git stash list
   ```
4. Apply the stashed changes:  
   ```bash
   git stash apply
   ```
5. Drop the stash after applying:  
   ```bash
   git stash drop
   ```

# **Task 6: Understanding the Git Workflow** 
   1. Make changes to a file in the **working directory**:  
      ```bash
      echo "Workflow example" > workflow.md
      ```
   2. Stage the file:  
      ```bash
      git add workflow.md
      ```
   3. Commit the file to the **repository**:  
      ```bash
      git commit -m "Added workflow example"
      ```

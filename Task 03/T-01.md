# **Task 1: Create a Local Git Repository**
1. Create a new project folder:  
   ```bash
   mkdir MyProject
   cd MyProject
   ```
2. Initialize it as a Git repository:  
   ```bash
   git init
   ```
   3. Verify the repository status:  
   ```bash
   git status
   ```

# **Task 2: Apply and Drop Stashed Changes**
1. Apply the most recent stash:  
   ```bash
   git stash apply
   ```
2. Drop the most recent stash after applying it:  
   ```bash
   git stash drop
   ```

3. Alternatively, to apply a specific stash:  
   ```bash
   git stash apply stash@{1}
   ```
